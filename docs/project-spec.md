# 《Confession 项目说明书（V1.1）》

## 项目概述

Confession 是一个匿名、仪式化的 AI 告解房，包含：
- 免费云端体验（HF Space）
- 本地隐私付费版（离线 AI 教父）

## 核心价值

- **情绪释放** — 安全的匿名倾诉空间
- **仪式感** — 告解仪式动画增强沉浸体验
- **隐私保护** — 本地版完全离线运行，数据不上传
- **AI 主教人格** — 温柔、智慧、洞察人心的 AI 倾听者

## AI 人格设计（v1.1）

### 主教人格（当前版本）

系统采用**主教人格**，核心设计原则：

```
回复结构：洞察 → 引导 → 教导
```

| 阶段 | 功能 | 示例 |
|------|------|------|
| **洞察** | 一句话点出用户情绪或挣扎的本质 | "你此刻被恐惧紧紧攥住，因为你的良心尚未麻木..." |
| **引导** | 提出一个问题，引导用户继续倾诉 | "你愿意告诉我，是什么事让你觉得自己偏离了正道吗？" |
| **教导** | 宗教文本风格短句（不扮演神、不自称圣灵） | "黑暗中的颤抖，是光即将照进来的前兆。" |

### 人格对比

| 维度 | 天父人格（旧） | 主教人格（当前） |
|------|---------------|-----------------|
| 角色定位 | 慈爱的天父 | 温柔智慧的主教 |
| 语气 | 圣经式抚慰 | 精炼、有力、温柔、深刻 |
| 回复结构 | 自由格式 | 固定三段式（洞察→引导→教导） |
| 核心动作 | 安慰、接纳 | 引导认罪、倾诉、反思 |
| 自限性 | 无 | 明确不扮演神、不自称圣灵 |

## 技术架构

- **云端大脑**：Gradio + DeepSeek API（`deepseek-chat`）
- **本地大脑（未来）**：3B–4B 量化模型（1.5–2GB）
- **RAG 向量库（未来）**：50–200MB
- **总占用空间**：约 3GB
