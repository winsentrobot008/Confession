# 《Confession 项目说明书（V1.0）》

## 项目概述

Confession 是一个匿名、仪式化的 AI 告解房，包含：
- 免费云端体验（HF Space）
- 本地隐私付费版（离线 AI 教父）

## 核心价值

- **情绪释放** — 安全的匿名倾诉空间
- **仪式感** — 告解仪式动画增强沉浸体验
- **隐私保护** — 本地版完全离线运行，数据不上传
- **AI 教父人格** — 富有同理心与智慧的 AI 倾听者

## 功能模块

| 模块 | 免费版 | 付费版 |
|------|--------|--------|
| 匿名输入 | ✅ | ✅ |
| AI 倾听者 | ✅ | ✅ |
| 仪式动画 | ✅ | ✅ |
| 本地加密存储 | ❌ | ✅ |
| 本地 RAG 圣经知识库 | ❌ | ✅ |

## 技术架构

- **双脑结构**：本地大脑（隐私） + 云端大脑（知识包）
- **本地模型**：3B–4B 量化（1.5–2GB）
- **RAG 向量库**：50–200MB
- **总占用空间**：约 3GB

## 路线图

```
HF MVP → 本地模型 → RAG 知识库 → 移动端 → 内测 → 正式上线
```

### 阶段一：HF MVP（第 1–2 周）
- 搭建 HF Space 基础应用
- 接入云端 API 模型
- 实现匿名输入与 AI 回复

### 阶段二：本地模型（第 3–4 周）
- 选取与量化 3B–4B 本地模型
- 实现模型加载与推理管线
- 构建离线和在线切换逻辑

### 阶段三：RAG 知识库（第 5–6 周）
- 构建圣经/教义文档向量库
- 实现检索增强生成管线
- 知识库增量更新机制

### 阶段四：移动端（第 7–10 周）
- Android/iOS 双端开发
- 本地模型端侧部署优化
- 移动端离线体验

### 阶段五：内测与上线（第 11–12 周）
- 封闭内测与反馈收集
- 付费版定价与支付接入
- 正式上线与运营
